					Read Me !

* MySQL DB file is named as pmstc.sql in the current folder. pmstc.sql file need to be imported into a MySQL DB created as   "pmstc".

* User login credentials are stored as Login_Credentials.txt with the assigned user roles.


To run the application, follow below instructions.
1. Download and install latest python on local machine. 	
	(https://www.python.org/downloads/)


2. To install the dependanices execute:
	pip install -r requirements.txt


3. To run the python programme:
	python ./run.py


4. Navigate to the web interface, run below URL on any web browser:
	127.0.0.1:5000
